异常处理

  1. 定义：异常指的是一个事件，该事件在程序执行过程中发生，影响程序的正常执行。（当python脚本发生异常时，我们需要捕获处理它，否则程序会中止执行）

  2. 异常处理机制：
    1）try 的工作原理：当开始一个try语句之后，python就会在当前程序的上下文做标记，这样当异常出现时，就会回到这里，try语句先执行，接下来执行什么取决于在执行的过程中是否出现异常。

    2） except: 当 try 里面的代码出现异常或者出错时，才会执行 except 里面的代码。
    只有except 时，对所有的错误都会进行处理
    当except + 具体的报错名称 时，·只会对报错名称的错误进行处理。

    3） else : 判断是否有异常，如果没有，执行此处的代码。

    4）finally ： 不论是否有异常，都会执行的代码。
